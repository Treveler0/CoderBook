Оформление HTML документов с помощью CSS

CSS был представлен вместе с HTML 4. CSS предоставляет более удобный способ оформления HTML документов.

CSS предоставляет полную свободу при оформлении документов, с CSS Вы можете:

- Устанавливать размер, начертание и цвет шрифта;

- Изменять местоположение элементов;

- Оформлять фон элементов;

- Выравнивать текст;

- Оформлять таблицы и списки и многое другое.

		<p style='font-size: 22px; display: inline;'>
			Это абзац оформленный с помощью CSS
		</p>
		<p style='display: inline; color: #f52f2f;'>
			Это еще один абзац оформленный с помощью CSS
		</p>

Устаревшие теги и атрибуты

После введения CSS некоторые теги и атрибуты в HTML стали считаться устаревшими (с помощью CSS можно добиться большего гораздо более эффективно). Они не будут поддерживаться в будущих версиях HTML и XHTML.

Избегайте использования следующих устаревших HTML тегов и атрибутов:

Теги Описание
<strike> Определяет зачеркнутый текст.
<font> и <basefont> Устанавливает шрифт для текста.
<center> Выравнивает содержимое по центру.
<menu> Определяет список меню.


Атрибуты Описание
color Определяет цвет текста.
bgcolor Определяет цвет фона.
align Определяет выравнивание текста.
