Спецсимволы HTML

Символы, которые невозможно отобразить в HTML документах обычным образом называются спецсимволами.

Спецсимвол можно отобразить только используя соответствующую ему мнемонику или специальный код.

Синтаксис:

		&мнемоника_символа;
		
		&#код_символа;

Использование спецсимволов

Самый часто используемый спецсимвол - это неразрывный пробел (его мнемоника &nbsp;).

Данный символ используется для контролирования переноса строки (после данного символа автоматический перевод строки невозможен) и для вставки в текст подряд идущих пробелов (по умолчанию если Вы вставите в текст 5 подряд идущих пробелов браузер вырежет 4 и отобразит лишь один):

		
Текст ниже содержит шесть пробелов: 

		
Первое слово &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; второе слово
Результат работы кода:

Текст ниже содержит шесть пробелов:

Первое слово             второе слово

Также в HTML коде часто используются спецсимволы < (мнемоника &lt;) и > (мнемоника &gt;).

Данные символы невозможно напрямую использовать в коде потому, что браузер путает их с началами тегов.

		
Текст ниже содержит HTML теги:

		
		&lt;head&gt;, &lt;p&gt;, &lt;html&gt;
Результат работы кода:

Текст ниже содержит HTML теги:

<head>, <p>, <html>
