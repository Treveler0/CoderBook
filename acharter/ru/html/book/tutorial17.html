Элемент script

HTML тег <script> используется для определения скриптов выполняющихся на стороне клиента. JavaScript является самым популярным языком программирования этого типа.

JavaScript используется для создания веб-приложений и динамических сайтов способных взаимодействовать с пользователем.

Тег script может либо содержать непосредственно код скрипта, либо ссылаться на внешний скриптовый файл через атрибут src.

Обратите внимание: в HTML5 атрибут type стал необязательным.

		<p>Нажмите на кнопку ниже, чтобы заданный скрипт выполнился.</p>
		
		<input type='button' value='Нажмите на меня' onclick='showMessage()' />
		
		<script type="text/javascript">
			function showMessage()
			{
				alert('Вы нажали на кнопку.')
			}
		</script>

Результат:

Нажмите на кнопку ниже, чтобы заданный скрипт выполнился.

В следующем примере элемент <script> ссылается на внешний скриптовый файл выводящий сообщение "Привет, привет!":

		<script type="text/javascript" src="hello.js"></script>

Элемент noscript

Тег <noscript> используется для отображения альтернативного текста пользователям, браузер которых не поддерживает JavaScript.

		<script type="text/javascript">
			document.write("<p>Привет, привет!</p>")
		</script>
		
		<noscript>
			<p>Вы видите это сообщение, потому что Ваш браузер не поддерживает JavaScript</p>
		</noscript>

Обратите внимание: узнать больше о JavaScript Вы можете в нашем JavaScript учебнике (рекомендуем переходить к изучению JavaScript только после изучения HTML).
