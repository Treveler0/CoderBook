Тег <link /> 

		<head>
			<link rel="stylesheet" type="text/css" href="style.css" />
		</head>

Определение и Использование

Тег <link> позволяет связать данный документ с внешним источником.

Тег <link> в основном используют для привязки внешних документов стиля. Данный тег должен определяться в head секции документа.

Атрибуты

Тег <link> поддерживает следующие атрибуты:

Атрибут Значение Описание
href URL Указывает URL внешнего документа.
rel alternate
appendix
bookmark
chapter
contents
copyright
glossary
help
home
index
next
prev
section
start
stylesheet
subsection		 Устанавливает тип отношений между текущим документом и внешним.
type Mime-тип Указывает mime тип ссылаемого элемента

Стандартные атрибуты

Тег <link> поддерживает следующие стандартные атрибуты:

Атрибут Значение Описание
class имя класса Определяет имя класса для данного элемента.
dir rtl
ltr		 Определяет направление текста в элементе.
id id Определяет уникальный id для элемента.
style определение стиля Определяет строковый стиль для элемента.
title текст Определяет дополнительную информацию об элементе.

Атрибуты событий

Тег <link> поддерживает следующие атрибуты событий:

Атрибут Описание
onclick Скрипт будет запущен по щелчку мыши.
ondblclick Скрипт будет запущен по двойному щелчку мыши.
onmousedown Скрипт будет запущен, когда кнопка мыши будет нажата.
onmousemove Скрипт будет запущен, когда указатель мыши будет перемещен в границах элемента.
onmouseout Скрипт будет запущен, когда указатель мыши выйдет за пределы границ элемента.
onmouseover Скрипт будет запущен, когда указатель мыши будет наведен на элемент.
onmouseup Скрипт будет запущен, когда нажатая кнопка мыши будет отпущена.
onkeydown Скрипт будет запущен, когда любая клавиша клавиатуры будет нажата.
onkeypress Скрипт будет запущен, когда клавиша клавиатуры будет нажата.
onkeyup Скрипт будет запущен, когда нажатая клавиша клавиатуры будет отпущена.
