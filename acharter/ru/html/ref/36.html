Определение и Использование

Тег <input> используется для сбора пользовательской информации в формах.

В зависимости от значения атрибута "type" данный тег значительно изменяет свое поведение, к примеру он может принимать вид: текстового поля, радио-кнопки, кнопки отправления, поля для ввода пароля и т.д.

Атрибуты

Тег <input> имеет следующие атрибуты:

Атрибут Значение Описание
alt текст Устанавливает альтернативный текст картинки (только для type="image").
checked checked Определяет этот элемент выбранным по умолчанию (только для type="checkbox" и type="radio").
disabled disabled Указывает, что элемент должен быть неработоспособным.
maxlength число Устанавливает максимальную длину поля в символах (только для type="text" и type="password").
name имя Указывает имя элемента input.
readonly readonly Указывает, что это поле должно использоваться только для чтения т.е. его нельзя редактировать (только для type="text" и type="password").
size число Указывает ширину элемента в символах.
src URL Указывает URL картинки которая будет отображаться как кнопка отправления.

Атрибуты событий

Тег <input> имеет следующие атрибуты событий:

Атрибут Описание
onblur Скрипт будет запущен когда элемент перестанет быть активным
onchange Скрипт будет запущен когда содержимое элемента изменится
onclick Скрипт будет запущен по щелчку мыши.
ondblclick Скрипт будет запущен по двойному щелчку мыши.
onfocus Скрипт будет запущен когда элемент станет активным
onmousedown Скрипт будет запущен, когда кнопка мыши будет нажата.
onmousemove Скрипт будет запущен, когда указатель мыши будет перемещен в границах элемента.
onmouseout Скрипт будет запущен, когда указатель мыши выйдет за пределы границ элемента.
onmouseover Скрипт будет запущен, когда указатель мыши будет наведен на элемент.
onmouseup Скрипт будет запущен, когда нажатая кнопка мыши будет отпущена.
onkeydown Скрипт будет запущен, когда любая клавиша клавиатуры будет нажата.
onkeypress Скрипт будет запущен, когда клавиша клавиатуры будет нажата.
onkeyup Скрипт будет запущен, когда нажатая клавиша клавиатуры будет отпущена.
