Определение и Использование

Тег <script> позволяет определить скрипт выполняющийся на стороне браузера.

Элемент может либо непосредственно содержать код, либо ссылаться на внешний файл содержащий код скрипта через атрибут src.

В атрибуте type указывается MIME тип скрипта (для JavaScript type=text/javascript).

Атрибуты

Тег <script> имеет следующие атрибуты:

Атрибут Значение Описание
type MIME тип Указывает mime тип скрипта.
charset кодировка Указывает кодировку использующуюся в файле скрипта.
src URL Указывает адрес расположения внешнего файла скрипта.

Атрибуты событий

Тег <script> не поддерживает ни одного атрибута событий.
