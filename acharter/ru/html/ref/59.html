Результат работы кода:
Атрибут Определение
table Определяет HTML таблицу
tr Определяет строку в таблице
td Определяет табличную ячейку
th Определяет табличный заголовок

Определение и Использование

С помощью тега <table> Вы можете определять таблицы.

Данные в таблицах могут разделятся на строки с помощью тега <tr>. Каждая строка в свою очередь может разделяться на ячейки с помощью тега

Таблицы также могут иметь табличный заголовок, который определяется с помощью тега <th>.

Cтандартные атрибуты

Тег <table> имеет следующие стандартные атрибуты:

Атрибут Значение Описание
border пиксели Указывает ширину таблицы в пикселях.
cellpadding пиксели Определяет отступ между границей ячейки и ее содержимым.
cellspacing пиксели Определяет величину отступа между ячейками таблицы.
width пиксели
%		 Указывает ширину таблицы.

Атрибуты событий

Тег <table> поддерживает следующие атрибуты событий:

Атрибут Описание
onclick Скрипт будет запущен по щелчку мыши.
ondblclick Скрипт будет запущен по двойному щелчку мыши.
onmousedown Скрипт будет запущен, когда кнопка мыши будет нажата.
onmousemove Скрипт будет запущен, когда указатель мыши будет перемещен в границах элемента.
onmouseout Скрипт будет запущен, когда указатель мыши выйдет за пределы границ элемента.
onmouseover Скрипт будет запущен, когда указатель мыши будет наведен на элемент.
onmouseup Скрипт будет запущен, когда нажатая кнопка мыши будет отпущена.
onkeydown Скрипт будет запущен, когда любая клавиша клавиатуры будет нажата.
onkeypress Скрипт будет запущен, когда клавиша клавиатуры будет нажата.
onkeyup Скрипт будет запущен, когда нажатая клавиша клавиатуры будет отпущена.
