Тег <meter>

Определение и Использование

Тег <meter> используется для отображения количественной меры чего-либо в ограниченном диапазоне.

Например данный тег может использоваться для отображения: количества занятого места на жестком диске, количество голосов за определенного кандидата и т.д.

Тег <meter> не должен использоваться для создания полосы загрузки (используйте тег <progress>) или отображения количественно меры чего-либо в неизвестном диапазоне (к примеру Вы не можете отобразить долю изученных человеком планет так как общее их количество неизвестно).

Атрибуты

Атрибут Значение Описание
form id формы Указывает форму, которой принадлежит данный элемент (элемент может также принадлежать к нескольким формам).
high число Указывает отрезок диапазона, значения которого должны считаться высокими.
low число Указывает отрезок диапазона, значения которого должны считаться низкими.
min число Указывает минимальное значение диапазона.
max число Указывает максимальное значение диапазона.
optimum число Указывает оптимальное значение диапазона.
value число Устанавливает текущее значение диапазона.
