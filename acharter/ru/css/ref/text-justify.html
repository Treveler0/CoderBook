CSS свойство text-justify 

		#el1
        {
		  text-align: justify;
		  text-justify: distribute;
		}

Описание

Обратите внимание: данное свойство поддерживается только в браузере Internet Explorer.

С помощью свойства text-justify Вы можете установить алгоритм, который будет использоваться для выравнивания текста методом justify (text-align:justify).

Данное свойство может принимать следующие значения:

  • auto браузер будет автоматически подбирать алгоритм выравнивания (является значением по умолчанию);
  • distribute для выравнивания браузер будет увеличивать или уменьшать расстояния между словами и буквами текста;
  • inter-word для выравнивания браузер будет увеличивать или уменьшать расстояние только между словами;
  • kashida для выравнивания браузер будет растягивать символы текста;
  • none браузер не будет выравнивать текст.
