CSS свойство border-left-style 

		/* Левая граница элемента будет отображена сплошной линией зеленого цвета толщиной 2 пикселя */
		
		p
		{
			border-left-style:solid;
			border-left-color:green;
			border-left-width:2px;
		}

Описание

С помощью свойства border-left-style Вы можете установить стиль левой границы элемента.

Возможные значения

Значения Описание
solid Определяет сплошную границу.
none Убирает границу.
dotted Определяет точечную границу.
dashed Определяет пунктирную границу.
double Определяет двойную границу.
groove Определяет объемную вдавленную границу.
ridge Определяет объемную выпуклую границу.
