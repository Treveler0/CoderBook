CSS свойство quotes 

		q
        {
		  quotes:"<<" ">>" "„" "“";
		}

Описание

С помощью свойства quotes Вы можете установить внешний вид кавычек, которые будут вставлены тегом <q>.

Первое значение данного свойства устанавливает вид открывающей кавычки, а второе вид закрывающей кавычки.

Вторая пара кавычек переданная данному свойству в третьем и четвертом значении будет использоваться для вложенных кавычек (например в "Дмитрий 'Аллигатор' Панин" одинарные кавычки являются вложенными).

Виды кавычек:

Вид Описание
" Двойная кавычка. Данный вид кавычек используется в текстах набранных на компьютере.
' Одинарная кавычка. Данный вид кавычек используется для вложенных кавычек в текстах набранных на компьютере.
<< Французская открывающая кавычка. Традиционная для русского языка открывающая кавычка.
>> Французская закрывающая кавычка. Традиционная для русского языка закрывающая кавычка.
„ Немецкая открывающая кавычка. Традиционная для русского языка открывающая кавычка используемая для вложенных кавычек.
“ Немецкая закрывающая кавычка. Традиционная для русского языка закрывающая кавычка используемая для вложенных кавычек.
