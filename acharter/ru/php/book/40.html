Обработка данных в массивах PHP

PHP имеет множество функций для обработки данных в массивах.

Например, если нужно вычислить сумму всех элементов массив, то можно воспользоваться функцией array_sum:

		<?php
			$arr[1] = 31; 
			$arr[2] = 1; 
			$arr[3] = 94;
			$sum = array_sum($arr); 
			echo $sum; 
		?>
Отображение в браузере:

126

Если, требуется избавится от элементов с повторяющим значение, то стоит использовать функцию array_unique:

		<?php
			$arr = array(30, 44, 97, 30);
			print_r($arr); 
			$new_arr = array_unique($arr);
			print_r($new_arr); 
		?>
Отображение в браузере:

Array ( [0] => 30 [1] => 44 [2] => 97 [3] => 30 )

Array ( [0] => 30 [1] => 44 [2] => 97 )
