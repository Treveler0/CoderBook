Функции для обработки строк в PHP

Начиная с этого урока мы с вами открываем главу посвященную функциям для обработки строковых данных.

С помощью этих фукнций можно, например, обрезать, дописать или заменить часть строки и много другое.

Это очень полезный инструмент и вы частенько будете использовать все эти функции при разработке свои скриптов.

Все функции для обработки строк перечислены ниже:
