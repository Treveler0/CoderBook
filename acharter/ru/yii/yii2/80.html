Генерация кода с Gii

Это расширение предоставляет генератор кода на основе Web, называемый Gii, для приложений Yii 2.

Вы можете использовать Gii для быстрого создания моделей, форм, модулей, CRUD и т. Д.

Gii предоставляет веб-интерфейс для интерактивного создания кода, который вы хотите.

Он также предоставляет интерфейс командной строки для людей, которые предпочитают работать со своими консольными окнами большую часть времени.

Получение с помощью Composer

Предпочтительный способ установки расширения через composer.

Для этого запустите

    php composer.phar require --dev --prefer-dist yiisoft/yii2-gii

или добавьте

    "yiisoft/yii2-gii": "~2.0.0"

в секцию require вашего composer.json.
