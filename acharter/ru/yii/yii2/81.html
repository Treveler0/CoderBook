Генератор документации API

Это расширение предоставляет генератор документации API для платформы Yii 2.0.

Установка

Предпочтительный способ установки расширения через composer.

Для этого запустите

    composer require --prefer-dist yiisoft/yii2-apidoc

или добавьте

    "yiisoft/yii2-apidoc": "~2.1.0"

в секцию require вашего composer.json.

Базовое использование

Это расширение предлагает две команды:

  • api для создания документации API класса.
  • руководство для рендеринга страниц HTML из файлов разметки, таких как руководство yii.

Простое использование для отдельной документации класса:

    vendor/bin/apidoc api source/directory ./output

Простое использование для самостоятельной документации руководства:

    vendor/bin/apidoc guide source/docs ./output

Вы можете комбинировать их для генерации API классов и руководства в одном месте:

    # generate API docs
    vendor/bin/apidoc api source/directory ./output
    # generate the guide (order is important to allow the guide to link to the apidoc)
    vendor/bin/apidoc guide source/docs ./output

По умолчанию будет использоваться шаблон начальной загрузки.

Вы можете выбрать другой шаблон с параметром --template=name.

В настоящее время доступен только шаблон начальной загрузки.

Вы также можете добавить yii\apidoc\commands\ApiController и GuideController в свою командную карту консольного приложения и запустить их внутри приложения приложений.
