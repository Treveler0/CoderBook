Объект Date

Объект Date используется для проведения различных манипуляций с датой и временем.

Подробную информацию о данном объекте Вы можете найти в нашем JavaScript учебнике.

Синтаксис:

		date=new Date();
		date=new Date(год, месяц, день, час, минута, секунда, миллисекунда);
		date=new Date(миллисекунды);

Свойства объекта Date

Свойство Описание
prototype Позволяет добавлять свойства и методы к объекту.

Методы объекта Date

Метод Описание
getDate() Возвращает день месяца (может принимать значения от 1-31) заданной даты.
getDay() Возвращает день недели (может принимать значения от 0-6, причем 0-Воскресенье, а 6-Суббота) заданной даты.
getFullYear() Возвращает год (4 числа, например 2011) заданной даты.
getHours() Возвращает час (может принимать значения от 0-23) заданной даты.
getMiliseconds() Возвращает миллисекунду (может принимать значения от 0-999) заданной даты.
getMinutes() Возвращает минуту (может принимать значения от 0-59) заданной даты.
getMonth() Возвращает месяц (может принимать значения от 0-11) заданной даты.
getSeconds() Возвращает секунду (может принимать значения от 0-59) заданной даты.
getTime() Возвращает количество миллисекунд, которые прошли с полночи 1 Января 1970 года до заданной в объекте даты.
getTimezoneOffset() Возвращает разницу текущего времени от Гринвича (GMT) в минутах.
getUTCDate() Возвращает день месяца (может принимать значения от 0-31) заданной даты по всемирному времени.
getUTCDay() Возвращает день недели (может принимать значения от 0-6, причем 0-Воскресенье, а 6-Суббота) заданной даты по всемирному времени.
getUTCFullYear() Возвращает год (4 числа, например 2011) заданной даты по всемирному времени.
getUTCHours() Возвращает час (может принимать значения от 0-23) заданной даты по всемирному времени.
getUTCMiliseconds() Возвращает миллисекунду (может принимать значения от 0-999) заданной даты по всемирному времени.
getUTCMinutes() Возвращает минуту, заданную в объекте, по всемирному времени (может принимать значения от 0-59).
getUTCMonth() Возвращает месяц (может принимать значения от 0-11) заданной даты по всемирному времени.
getUTCSeconds() Возвращает секунду (может принимать значения от 0-59) заданной даты по всемирному времени.
parse() Возвращает количество прошедших с полночи 1 Января 1970 года миллисекунд до заданной даты.
setDate() Устанавливает день месяца (может принимать значения от 0-31) заданной дате.
setFullYear() Устанавливает год (4 числа, например 2011) заданной дате.
setHours() Устанавливает час (может принимать значения от 0-23) заданной дате.
setMiliseconds() Устанавливает миллисекунду (может принимать значения от 0-999) заданной дате.
setMinutes() Устанавливает минуту (может принимать значения от 0-59) заданной дате.
setMonth() Устанавливает месяц (может принимать значения от 0-11) заданной дате.
setSeconds() Устанавливает секунду (может принимать значения от 0-59) заданной дате.
setTime() Устанавливает дату и время путем прибавления или вычитания указанного количества миллисекунд от полночи 1 Января 1970.
setUTCDate() Устанавливает день месяца по всемирному времени (может принимать значения от 0-31) заданной дате.
setUTCFullYear() Устанавливает год по всемирному времени (4 числа, например 2011) заданной дате.
setUTCHours() Устанавливает час по всемирному времени (может принимать значения от 0-23) заданной дате.
setUTCMilliseconds() Устанавливает миллисекунду по всемирному времени (может принимать значения от 0-999) заданной дате.
setUTCMinutes() Устанавливает минуту по всемирному времени (может принимать значения от 0-59) заданной дате.
setUTCMonth() Устанавливает месяц по всемирному времени (может принимать значения от 0-11) заданной дате.
setUTCSeconds() Устанавливает секунду по всемирному времени (может принимать значения от 0-59) заданной дате.
toDateString() Преобразует часть объекта содержащую дату в строку.
toString() Преобразует объект в строку.
toTimeString() Преобразует часть объекта содержащую время в строку.
toUTCString() Преобразует объект в строку по всемирному времени.
UTC() Возвращает количество прошедших с полночи 1 Января 1970 миллисекунд до указанной даты по всемирному времени.
