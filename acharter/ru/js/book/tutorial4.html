JavaScript команды

С помощью JavaScript команд Вы можете "общаться" с браузером.

Например с помощью команды document.write('Привет!'); Вы можете сообщить браузеру, что хотите вывести на страницу текст "Привет!".

Точка с запятой сообщает браузеру о конце команды.

Обратите внимание: в JavaScript необязательно явно указывать точку запятой после каждой команды, так как браузер автоматически подставит ее если следующая команда будет написана на новой строке.

В отличие от HTML JavaScript чувствителен к регистру букв.

Всегда обращайте внимание на регистр букв при наборе команд в JavaScript.

		document.wrITE("<hr /><b>Поздравляем, Вы исправили ошибку! </b>");

JavaScript код

JavaScript код является последовательностью JavaScript команд.

Команды исполняются браузером сверху вниз.

Следующий пример выводит два абзаца, написанных жирным и курсивным текстом, на страницу:

		document.write("<p><b>Это абзац написанный жирным шрифтом.</b></p>");
		document.write("<p><i>Это абзац написанный курсивным шрифтом.</i></p>");

JavaScript блоки команд

JavaScript команды могут объединяться в блоки.

Обычно блоки используют при определении функций, создании циклов и в некоторых других случаях, которые будут рассмотрены далее в данном учебнике.

Начало блока команд обозначается открывающейся фигурной скобкой {, а конец закрывающейся }.

Пример использования блоков:

		function mesg()
		{
		  document.write("<p><b>Это абзац написанный жирным шрифтом.</b></p>");
		  document.write("<p><i>Это абзац написанный курсивным шрифтом.</i></p>");
		}
