Функции для манипулирования числами

Числовые функции предназначены для манипулирования числовыми данными.

Эти функции используются только для алгебраических, тригонометрических и геометрических вычислений, поэтому они используются не так часто, как функции манипулирования датой и временем.

По иронии судьбы среди всех функций в большинстве СУБД именно числовые функции наиболее стандартизированы.

В таблице ниже перечислены наиболее часто используемые функции манипулирования числовыми данными.

Функция Описание
ABS () Возвращает абсолютное значение числа
COS () Возвращает косинус указанного угла
ЕХР () Возвращает экспоненту указанного числа
PI () Возвращает значение числа пи
SIN () Возвращает синус указанного угла
SQRT () Возвращает квадратный корень указанного числа
TAN () Возвращает тангенс указанного угла

Обратитесь к документации по вашей СУБД, чтобы определить, какие функции манипулирования числовыми данными она поддерживает.
